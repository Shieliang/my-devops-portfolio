SESB Intelligent Bot: 跨语言 RAG 智能客服系统

Generative AI RAG

技术栈： AWS Bedrock (Claude 3.5 Sonnet, Titan Embeddings), AWS EC2, S3, LangChain, FAISS, Streamlit

项目描述： 为解决本地电力服务申请过程中的多语言支持痛点，我开发了这款基于生成式 AI 的智能客服。系统利用 RAG 技术读取 S3 中的官方英文/马来文 PDF 指南，并能用精准的中文解答用户的复杂疑问，同时提供所需表格的 S3 预签名下载链接。

SL Photo Converter: $0 成本 Serverless 图像转换器

Serverless Architecture

技术栈： AWS S3, AWS Lambda (Python + Pillow Layer), Amazon API Gateway, CloudFront

项目描述： 告别 24/7 空转的服务器！这是一个完全基于事件驱动 (Event-Driven) 架构的图片格式转换工具。用户通过前端获取预签名 URL 直接将图片安全上传至 S3，自动触发 Lambda 函数进行格式转换，并将结果输出至目标存储桶。

核心亮点：

  • 极致的成本优化： 利用 AWS Free Tier 和 S3 Lifecycle Rules (24小时自动清理)，实现了项目的 $0 运行成本。
  • 安全性设计： 采用 CloudFront OAC 机制和严格的 CORS 配置，确保源站绝对私密与安全。
  • 轻量化计算： 独立编译打包 Pillow 依赖为 Lambda Layer，优化函数冷启动与部署体积。

基于 IaC 与 CI/CD 的云端作品集网站 (本项目)

DevOps SecOps

项目描述： 本作品集网站的基础设施完全由 Terraform 声明式代码 (IaC) 部署。从零打通了代码到云端的持续集成与交付，并融入了企业级的云安全最佳实践与数据可观测性矩阵。
